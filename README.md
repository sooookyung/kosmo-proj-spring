# 기능소개
* 메인 화면에는 음식 이미지 슬라이드쇼 구현
* 사이트 소개 페이지 구현
* 게시판/댓글 기능 구현
* 게시판 카테고리별 분류 기능 구현
* 지도API로 맛집 정보의 위치 표시 기능 구현
* 로그인/회원가입/로그인 유지
